# RachaAí Blazor

## Implementação Blazor

A hierarquia visual criada no Miro foi aplicada utilizando cards em destaque para os valores principais e componentes reutilizáveis para os grupos.

As cores verde e vermelha ajudam o usuário a identificar rapidamente créditos e dívidas.

## Dificuldade Técnica

O maior desafio foi componentizar o GrupoCard para reutilizar o layout dos grupos sem repetir código.
